## 📚 Kurs dasturi

Quyida 11 ta modul oson'dan murakkabga qarab, mantiqiy ketma-ketlikda tuzilgan. Har bir mavzu o'zidan oldingisiga asoslanadi.

---

### 🔹 Modul 0 — Tayyorgarlik va vositalar
> *Maqsad: ish muhitini sozlash va birinchi dasturni yozish.*

- [ ] Dasturlash nima va kompyuter qanday "o'ylaydi"
- [ ] **uv** bilan Python o'rnatish va versiyalarni boshqarish (zamonaviy, tez vosita)
- [ ] VSCode o'rnatish va sozlash (extensions, sozlamalar)
- [ ] Terminal / CLI asoslari (`cd`, `ls`, `mkdir`, fayl boshqarish)
- [ ] Birinchi dastur — "Hello, World!"
- [ ] `print()` va `input()` bilan ishlash

📅 **2 dars**

---

### 🔹 Modul 1 — Asoslar
> *Maqsad: ma'lumot bilan ishlashning poydevorini qo'yish.*

- [ ] O'zgaruvchilar (variables) va ularni nomlash qoidalari
- [ ] Ma'lumot turlari: `int`, `float`, `str`, `bool`
- [ ] Type conversion (turlar o'rtasida o'tkazish)
- [ ] Operatorlar: arifmetik, taqqoslash, mantiqiy
- [ ] f-strings bilan matnni formatlash
- [ ] Kommentariyalar va kodni o'qiluvchan qilish

📅 **3 dars**

---

### 🔹 Modul 2 — Shartlar va sikllar
> *Maqsad: dasturga "qaror qabul qilish" va "takrorlash" qobiliyatini berish.*

- [ ] `if / elif / else` — shartli ifodalar
- [ ] Mantiqiy ifodalar (`and`, `or`, `not`)
- [ ] `for` sikli va `range()`
- [ ] `while` sikli
- [ ] `break`, `continue`, `pass`
- [ ] Nested loops (ichma-ich sikllar)

📅 **3 dars**

---

### 🔹 Modul 3 — Ma'lumotlar strukturalari
> *Maqsad: ko'p ma'lumotni samarali saqlash va boshqarish.*

- [ ] String va uning metodlari, slicing
- [ ] **List** — eng ko'p ishlatiladigan struktura
- [ ] **Tuple** — o'zgarmas ketma-ketlik
- [ ] **Dictionary** — key-value juftliklari
- [ ] **Set** — takrorlanmas elementlar

📅 **4 dars**

---

### 🔹 Modul 4 — Funksiyalar
> *Maqsad: kodni qayta ishlatiladigan bloklarga bo'lish.*

- [ ] `def` — funksiya yaratish, parametr va argument
- [ ] `return` bilan natija qaytarish
- [ ] Default va keyword arguments
- [ ] `*args` va `**kwargs`
- [ ] Scope (local / global o'zgaruvchilar)
- [ ] `lambda` funksiyalari
- [ ] Recursion (rekursiya) — kirish

📅 **3 dars**

---

### 🔹 Modul 5 — Paketlar, muhit va modullar
> *Maqsad: tashqi kutubxonalar va izolyatsiyalangan muhit bilan ishlash.*

- [ ] **Virtual environment (venv)** — nima va nega kerak
- [ ] **uv** bilan loyiha boshqaruvi: `uv init`, `uv add`, `uv run`

### 🔹 Modul 6 — Xatoliklar va debugging
> *Maqsad: xatoliklardan qo'rqmaslik va ularni topa olish.*

- [ ] `raise` — xatolikni o'zi chaqirish
